# Portfolio Entry: Sweet Home Real Estate Platform

## Title
**Sweet Home Real Estate Agency Platform**

## Slug
`sweet-home-real-estate-platform`

## Description

Full-stack web platform for a real estate agency operating in Berlin, Dubai, and North Cyprus. Built with Node.js, Express, and PostgreSQL, the platform enables property listings, project management, lead tracking, and multi-language content (English, Spanish, German).

**Key Features:**
- Property and project listings with advanced search, filtering, and interactive maps
- Admin dashboard with analytics, content management, and lead tracking
- Multi-language support with i18n
- Media management with image processing and YouTube video integration
- Blog system and mortgage calculator
- Role-based access control for agents and administrators

**Tech Stack:** Node.js, Express, PostgreSQL, EJS, DigitalOcean Spaces, Chart.js
