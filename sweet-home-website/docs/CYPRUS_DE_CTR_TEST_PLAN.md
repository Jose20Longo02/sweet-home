# Cyprus DE CTR Test Plan

This document defines title/meta variants for `/de/immobilien-zypern-kaufen` to test in Google Search Console.

## Scope

- Page: `https://sweet-home.co.il/de/immobilien-zypern-kaufen`
- Language: German only
- Goal: Improve CTR for high-intent queries around:
  - `haus kaufen zypern`
  - `immobilien zypern`
  - `wohnung zypern kaufen`
  - `immobilien auf zypern kaufen`

## Testing protocol

- Use one variant at a time (no overlap).
- Keep each variant live for 14 days minimum.
- Compare by query in Search Console (same page + same query set).
- Decision metric:
  - Primary: CTR
  - Secondary: clicks (if impressions stable)
- Keep winner and test next challenger.

## Title variants

- **T1 (current baseline)**  
  `Haus kaufen Zypern: Immobilien & Wohnungen auf Zypern kaufen`

- **T2 (intent-first)**  
  `Immobilien Zypern kaufen: Haus oder Wohnung in Top-Lagen`

- **T3 (benefit-first)**  
  `Haus kaufen in Zypern: Gepruefte Immobilien in Paphos, Limassol, Larnaka`

## Meta description variants

- **M1 (current baseline)**  
  `Haus kaufen Zypern oder Wohnung Zypern kaufen: Entdecken Sie Immobilien auf Zypern in Paphos, Limassol, Larnaka und Nikosia mit professioneller Beratung von Sweet Home.`

- **M2 (decision support)**  
  `Immobilien auf Zypern kaufen leicht gemacht: Vergleichen Sie Haus- und Wohnungsangebote nach Lage, Preis und Potenzial - mit professioneller Begleitung von Sweet Home.`

- **M3 (action + trust)**  
  `Jetzt passende Zypern Immobilien finden: Haus kaufen in Zypern oder Wohnung kaufen in den gefragtesten Regionen. Transparente Beratung von der Suche bis zum Abschluss.`

## Suggested sequence

1. T1 + M1 (baseline snapshot 14 days)
2. T2 + M2 (challenger #1)
3. T3 + M3 (challenger #2)
4. Keep top performer
